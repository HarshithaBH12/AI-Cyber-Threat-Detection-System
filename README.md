# 🔐 AI Cyber Threat Detection System

This project is a multi-agent AI system designed to detect phishing and suspicious messages.

## 🚀 Features
- Multi-agent architecture
- Link analysis (real-world logic)
- Threat scoring
- Threat classification (Phishing, Spam)
- Confidence calculation
- Explainable output
- Response system

## 🧠 Architecture
Input → Agents → Merge → Decision → Response

## 🤖 Agents
- Urgency Detector
- Link Analyzer
- Tone Analyzer
- Spam Detector
- Repetition Detector
- Domain Checker

## 📊 Example Output
🔴 High Risk | Type: Phishing | Score: 9 | Confidence: 96%  
Reasons: Link detected, urgency language, spam keywords  
Action: Block message + Alert user

## ⚠️ Note
This system uses heuristic-based detection and can be extended with APIs like Google Safe Browsing.
